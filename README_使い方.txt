旅行サイト リンク版

使い方:
1. このZIPを解凍します。
2. index.html を開くとTOPページが表示されます。
3. TOPから山口ルートへ移動できます。
4. 山口ページから1日目詳細・2日目詳細へ移動できます。
5. Netlifyに上げる場合は、このZIPの中身をそのままアップロードしてください。

ファイル構成:
- index.html              TOPページ
- yamaguchi.html          山口工程ページ
- yamaguchi_day1.html     山口1日目詳細
- yamaguchi_day2.html     山口2日目詳細
- shikoku.html            四国ページ用の仮ページ

四国ページが完成したら:
- 完成した四国HTMLのファイル名を shikoku.html にして、この仮ページと置き換えてください。
