旅行サイト_レスポンシブ対応_リンク版

入っているファイル:
- index.html : TOPページ
- yamaguchi.html : 山口ルート工程ページ
- yamaguchi_day1.html : 山口1日目詳細
- yamaguchi_day2.html : 山口2日目詳細
- shikoku.html : 四国ルート準備中ページ
- password.html : パスワード入力用ページ

使い方:
ZIPを解凍して index.html を開くと通常のTOPページから確認できます。
四国ページが完成したら shikoku.html を差し替えれば、TOPページのリンクはそのまま使えます。
password.html は簡易的な入口ページです。HTMLだけのパスワードなので本格的な非公開化ではありません。
